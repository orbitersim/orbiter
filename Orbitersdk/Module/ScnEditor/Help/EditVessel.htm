Orbiter Scenario Editor

Configuring a vessel

After clicking the Edit button on the main page, or after creating a new vessel, the editor will display the vessel configuration page for the selected vessel. From here you can control vari­ous aspects of the vessel’s state, such as its position, velocity, orientation, and vessel-specific parameters.

Note: Although not strictly necessary, it is sometimes useful to pause the simulation (with Ctrl P or by pressing the "P" button in the title bar of the editor dialog) before editing a vessel. Otherwise the continuously changing state parameters may make a precise configuration tricky.

The following configuration options are currently available:

  • Orbital elements (freeflight mode)
  • State vectors (freeflight mode)
  • Vessel orientation (freeflight mode)
  • Angular velocity (freeflight mode)
  • Surface location (surface mode)
  • Propellant levels
  • Docking management
