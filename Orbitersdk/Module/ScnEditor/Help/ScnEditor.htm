Orbiter Scenario Editor

Scenario Editor

Orbiter has an editor that allows to create, configure and delete vessels within a running simulation. The editor is provided as a plugin module. To use it, make sure that the ScnEditor module is activated in the Modules tab of the Orbiter Launchpad dialog.

During the simulation, you can access the editor by opening the Custom Functions dialog with Ctrl F4, and double-clicking the Scenario Editor entry in the list. This will bring up the editor's main page, containing an inventory of all vessels in the current simula­tion session. You have now the option to:

  • create a new vessel
  • edit the vessel selected in the list
  • delete the selected vessel
  • change the simulation date
  • save a modified scenario to a file
