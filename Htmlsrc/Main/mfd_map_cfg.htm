Map MFD configuration

Scenario file options. Use inside a BEGIN_MFD/END_MFD block to define the MFD startup configuration.
ItemParametersNotes
TYPE Map MFD mode description
REF <Name> Reference body. The celestial body for which the map is displayed.
BTARGET <Base name> Target surface base. The target base is indicated in the map, and its position is shown above. The base must exist on the reference body. Default: no target.
OTARGET <Vessel/planet name> Name of the vessel or planet moon whose trajectory is to be displayed in the map. If defined, the object must exist and must be orbiting the reference body. Default: no target.
ZOOM If specified, the map is displayed in 2x zoom mode. Default: global view.
TRACK ON | <xpos> <ypos> Track mode. This is either ON (track mode enabled), or disabled, in which case the x and y coordinates of the map point in the top left corner of the display can be specified (x and y can range from 0 to 1). Default: no track mode.
