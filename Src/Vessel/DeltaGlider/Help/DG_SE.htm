Delta-glider - Orbiter Scenario Editor

Scenario Editor

The Orbiter Scenario Editor allows to modify and save many spacecraft parameters during the simulation. To access the editor, the ScnEditor must be activated in the Modules tab of the Orbiter Launchpad. The editor can then be accessed from the Custom functions list (Ctrl F4).
