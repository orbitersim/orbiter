Sync Orbit MFD

Controls Display components

The Synchronise Orbit MFD assists in catching up with an orbiting body. The instrument displays the ship's and target body's orbits, together with a reference axis and lists the times it will take both objects to reach this axis for a series of orbits.

Warning: For this instrument to work properly the orbital planes of both objects must coincide. The relative inclination of the orbital planes is shown in the lower left corner ("RInc"). If this becomes greater than 1°, realign the planes using the Align Orbital Planes MFD. Once the planes are aligned, all subsequent manoeuvres should be performed in this plane.
