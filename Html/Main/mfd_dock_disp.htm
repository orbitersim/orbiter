Docking MFD display

Move the mouse over the images to see a description of the different elements.

Telemetry source identifier for IDS or XPDR signal.
TOFS: Tangential offset from approach path, in units of the approach cone radius at the current target distance. TOFS < 1 indicates a position inside the approach cone.
TVEL: Tangential velocity (velocity relative to target, projected into plane normal to approach path) [m/s].
Signal acquisition mode: IDS (Instrument Docking System), XPDR (transponder), or VIS (visual).
Slaved NAV receiver.
NAV receiver frequency [kHz].
Distance to target (XPDR) or distance to dock (IDS, VIS) [m].
Closing speed (>0: approaching, <0: retreating).
Distance bar (logarithmic scale). Range: 0.1m - 1000m.
Closing speed bar (logarithmic scale). Range: 0.1m/s - 1000m/s. Yellow: closing, green: retreating.
Alignment of rotation around longitudinal axis (12 o'clock position indicates correct alignment). Only displayed if directional alignment is within 5°.
Approach path indicator. Indicates position of approach path relative to ship. When centered, the ship is aligned on approach path. Radial scale is logarithmic (0.1 - 1000m).
Alignment indicator: alignment of ship's forward direction with approach path direction. When centered, the ship's forward direction is parallel to the approach path. Radial scale is linear (0 - 20°). Indicator turns white if misalignment < 0.5°.
Tangential velocity indicator. Relative tangential velocity with respect to target. Radial scale is logarithmic (0.01 - 100 m/s).
Visual alignment indicator.
Legend:
Green/red circle: Approach cone. The concentric red or green circle indicates the size of the approach cone at the current dock distance. The ship should approach the dock so that the ap­proach path indicator is always inside the approach cone (indicated by a green circle). The approach cone becomes smaller as the ship approaches the dock.
