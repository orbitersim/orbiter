Annotation methods

The Annotation class contains methods for displaying and formatting onscreen annotations. Use oapi.create_annotation to retrieve an Annotation object.

In the list below, n indicates a valid annotation object.

See also:

oapi.create_annotation, oapi.del_annotation.

Methods:

n:set_text Sets the text to display.
n:set_pos Sets the screen position for the annotation.
n:set_size Sets the font size for the annotation.
n:set_colour Sets the font colour for the annotation.

n:set_text(str)

Sets the text to display by the annotation.

Parameters:

str (string):annotation string

n:set_pos(xmin,ymin,xmax,ymax)

Sets the screen position of the bounding box for the annotation.

Parameters:

xmin (number):coordinates of left edge
ymin (number):coordinates of top edge
xmax (number):coordinates of right edge
ymax (number):coordinates of bottom edge

Notes:

The coordinates of the annotation bounding box must be specified as fractions of the viewport size, in the range 0 to 1, where 0 represents the left and top viewport borders, and 1 represents the right and bottom viewport borders.

0 ≤ xmin < xmax ≤ 1 and 0 ≤ ymin < ymax ≤ 1 is required.

n:set_size(s)

Sets the font size for the annotation object.

Parameters:

s (number):relative font size (> 0)

Notes:

A size value of 1 represents the default font size. Values < 1 produce smaller fonts, values > 1 produce larger fonts.

The font size is automatically scaled with the viewport size.

n:set_colour(col)

Sets the font colour for the annotation object.

Parameters:

col (table):RGB colour values

Notes:

The col table must contain the fields 'r', 'g' and 'b' for the red, green and blue components, respectively.

Values for each component must be in the range from 0 to 1.

Example:

n:set_colour({r=1,g=0.7,b=0.3})
